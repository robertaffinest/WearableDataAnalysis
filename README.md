## Peer-graded Assignment: Getting and Cleaning Data Course Project

The purpose of this project is to demonstrate your ability to collect, work with, and clean a data set. The dataset is derived from accelerometer data collected by the Samsung Galaxy S smartphone. The goal is to create a tidy dataset that can be used for further analysis and insights into human activities.

## Repository Contents

- **run_analysis.R**: An R script that merges training and test datasets, extracts relevant measurements (mean and standard deviation), labels the activities and variables with descriptive names, and creates a tidy dataset with the average of each variable for each activity and subject.
- **CodeBook.md**: A detailed code book describing the variables in the dataset, the transformations performed, and relevant summaries.
- **README.md**: This file provides an overview of the project, instructions on how to run the analysis script, and explanations of the repository structure.

## Dataset Source

The dataset used in this project is obtained from the UCI Machine Learning Repository. For more information about the dataset, please visit: [UCI Human Activity Recognition Dataset](http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones).

## How to Run the Analysis

To run the analysis, ensure you have the necessary R packages installed, and execute the `run_analysis.R` script in RStudio. The resulting tidy dataset will be saved for further use.
